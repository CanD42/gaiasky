Gaia Orbit Applet

Welcome to the Gaia Orbit Applet, the Java Applet developed in the Astronomisches Rechen-Institut, The Univeristy of Heidelberg. Please be patient, it takes a while to load.

You need Java installed to play this game.
You may use the button below to get it.

Controls:
right mouse button - Pan the view
middle mouse button/wheel - Zoom the view
left mouse button - Rotate the view
i - Show coordinate axes
p - Pause simulation
Esc - Stop program
0 - Free camera
1 - Camera follows orbit
2 - Camera follows Gaia
3 - Camera is at a fixed position with respect to Gaia
4 - Camera rotates with Gaia
5 - Camera rotates and precesses with Gaia
6 - Camera inside Gaia
7 - Scene camera
8 - Focus camera, follows the selected object
